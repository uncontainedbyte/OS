a custom operating system built from scratch.
it is a personal hobby project, a learning experience, and a modern nod to the classic "just a hobby, wont be big and professional - linus" mindset.

the ultimate goal is to write the OS in **Zult**, a custom programming language im making(TempleOS style).
because i got impatient waiting for the Zult to be ready enough, so the initial kernel is being written in **C**.

## Current Status
- **Language:** Freestanding C (No standard library)
- **Target:** x86
- **Environment:** QEMU(hopefully hardware later)
